# LoRa-Based Landslide Detection and Early Warning System

## Overview

The **LoRa-Based Landslide Detection and Early Warning System** is an IoT-based project developed to monitor environmental conditions in landslide-prone regions. The system combines multiple environmental sensors, LoRa communication, Wi-Fi connectivity, and machine learning to detect hazardous conditions and provide real-time alerts.

This project demonstrates the integration of embedded systems, wireless communication, sensor fusion, and cloud-based monitoring for disaster prevention.

---

## Features

* Long-range LoRa wireless communication
* ESP32-based embedded system
* Multi-sensor environmental monitoring
* Machine Learning (Random Forest) prediction
* Wi-Fi cloud data logging
* LCD status display
* LED and buzzer alert system
* Low-power IoT architecture

---

## Hardware Components

* ESP32 Development Board
* LoRa SX1278 Module
* MPU6050 Accelerometer
* Soil Moisture Sensor
* Vibration Sensor
* DHT11 Temperature & Humidity Sensor
* 16×2 I2C LCD Display
* LEDs
* Buzzer
* Push Button
